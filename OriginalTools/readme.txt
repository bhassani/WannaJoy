CryptGenRandom tools
Author: Leo Dorrendorf

Synopsis
========
This sample includes tools to:
1. Extract the state of CryptGenRandom into a file
2. Simulate CryptGenRandom's next outputs from a saved state file
3. Simulate CryptGenRandom's previous outputs from a saved state file

Compiles with nmake.
Use run.bat to run the full example script, with output file comparisons.
Requires WinSDK installed to run.
